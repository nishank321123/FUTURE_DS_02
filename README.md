# Social Media Campaign Performance Analysis Dashboard

## Internship
Future Interns – Data Science & Analytics Track

## Task
Task 2 – Social Media Campaign Performance Tracker

## Objective
The objective of this project is to analyze marketing campaign data to understand customer spending behavior, campaign response rates, and purchasing patterns.

## Tools Used
- Power BI
- Data Cleaning
- Data Visualization
- DAX

## Dataset
Customer Marketing Campaign Dataset containing information about:
- Customer demographics
- Product spending
- Purchase channels
- Campaign responses

## Dashboard Insights
Key insights discovered from the analysis:

- Customers from Spain generate the highest revenue.
- Wine products contribute the largest portion of total spending.
- Most purchases occur through web channels.
- Customers with Graduation education respond more to marketing campaigns.

## Dashboard Components
The dashboard includes:

- KPI Cards
  - Total Spending
  - Total Purchases
  - Average Income
  - Campaign Response Rate

- Visualizations
  - Spending by Country
  - Product Category Sales
  - Purchase Channel Distribution
  - Campaign Response by Education
  - Age vs Spending Trend
